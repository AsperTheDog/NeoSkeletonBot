Command


This action will parse a string and divide the command and the arguments.
For example, given the following string:

.ban @user spam

The command will output:
- As main command -> ".ban"
- As arguments -> {0: "@user", 1: "spam"}

The command node will detect everything between quotes as a single argument.
That is, given the string

.ban @user "you spammed in general"

The command will output:
- As main command -> ".ban"
- As arguments -> {0: "@user", 1: "you spammed in general"}

It detects both simple and double quotes
