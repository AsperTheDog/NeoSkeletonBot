GetUser


Given a user ID, this action will return the data of the user.

Structure keys of a user are:
- id(number): The id of the user used as input
- avatar(text): Link to the profile picture image
- bot(boolean): True if the user is a bot, false otherwise
- banner(text): Link to the banner if the user has one, otherwise it's an empty text
- created date(datetime): Datetime of the moment when the account was created
- name(number): Name of the user
- nick(text): Nick of the user in the guild
- discriminator(text): Discriminator of the user
- identifier(text): The full username of the user in the format 'name#discriminator'
- roles(structure): A structure containing the list of roles where the id of each role is the key and the value is the name
- mention(text): A text piece in the format of a discord mention of this user
- top_role(number): The id of the highest role the user has in the guild
