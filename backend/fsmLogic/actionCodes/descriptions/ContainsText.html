ContainsText

This action will receive two texts and check if the first text contains the second one.

Some examples are:
- "hello" is contained in "hello how are you"
- "tex" is contained in "latex"
- "co" is contained in "Discord"

- "ga" is not contained in "guard"
- "CO" is not contained in "Discord"
- "dis" is not contained in "Discord"


If the second text is contained, the event "Contained" is triggered
If the second text is not contained, the event "Not contained" is triggered
