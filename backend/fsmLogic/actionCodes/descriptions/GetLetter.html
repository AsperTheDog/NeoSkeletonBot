GetLetter


Given a number and a text, this action will return the character present in the given text at the position specified in the given number.

The position is counted starting at 0, not at 1

Some examples:
- Given the text "example" and the number 3, the action will return "m"
- Given the text "hello" and the number 0, the action will return "h"
- Given the text "this is a test" and the number 8, the action will return "a"
- Given the text "oh my god" and the number 4, the action will return "y"
