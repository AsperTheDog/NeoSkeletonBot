CreateEmbed


This action generates a basic embed structure given some basic parameters This is the basic layout of an embed:



The url field will make the embed title be a link if filled. If you don't want the title to be a link, then don't connect this field to anything

The color field supports two formats:
One is a structure outputted by the actions createRGB and RGBFromHex
The other is a text that has the name of the color. The color name texts supported by the action are:
- ■ blue
- ■ blurple
- ■ brand_green
- ■ brand_red
- ■ dark_blue
- ■ dark_gold
- ■ dark_gray
- ■ dark_green
- ■ dark_grey
- ■ dark_magenta
- ■ dark_orange
- ■ dark_purple
- ■ dark_red
- ■ dark_teal
- ■ dark_theme
- ■ darker_gray
- ■ darker_grey
- ■ default
- ■ fuchsia
- ■ gold
- ■ green
- ■ greyple
- ■ light_gray
- ■ light_grey
- ■ lighter_gray
- ■ lighter_grey
- ■ magenta
- ■ og_blurple
- ■ old_blurple
- ■ orange
- ■ purple
- random
- ■ red
- ■ teal
- ■ yellow






Embed structure format:
- title: Title of the embed
- description: Description of the embed
- url: Link attached to the title of the embed
- color: Color to use in the embed
- fields: List of fields to be displayed in the embed

It is not recommended to extract the value of this structure since most manage data that is not compatible with SkeletonBot's interface.
The commands will simply give an error and refuse to execute until you change it.
